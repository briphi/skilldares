The target user for Skilldares is **new Kildares pub staff** — first-week / pre-shift-ready hires who need to learn the menu fast. Not veteran bartenders building expertise, not managers running training programs.

**Why:** During the 2026-05-22 brainstorming session, user explicitly narrowed scope from a multi-persona staff app down to persona 1 only ("First-Week Server Erin"). Confirmed mid-session after seeing ideas for veteran-bartender and manager personas.

**How to apply:** Frame features around the onboarding pain — pronunciation, allergen safety, section-mapping, dish-to-ingredient recall, confidence-building. Skip leaderboards across veterans, manager dashboards, train-the-trainer authoring, P&L reporting, etc. If a feature only makes sense for someone who already knows the menu, it's out of scope. See [[project-skilldares-vision]].
