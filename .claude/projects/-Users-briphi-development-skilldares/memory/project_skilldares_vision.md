Skilldares is a static web application whose product vision is to quiz Kildares (an Irish pub) staff on the Kildares menu. The name = "skill" + "Kildares".

**Why:** The repo's README says "A Kildares learning application" — the menu data files (`data/menu_front.md`, `data/menu_back.md`) are the source of truth that staff need to learn. The current `index.html` is a basic landing page; the staff-quiz app is the intended product, not customer-facing menu browsing.

**How to apply:** When working on Skilldares, treat the audience as Kildares pub staff (servers, bartenders, hosts, kitchen) — not pub patrons. The menu markdown files are the content source. Constraint: static web app, no backend.
